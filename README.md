# YAPP-
Yet Another Path Planner is a Path Planning tool for testing said algorithms in 2D using Qt6. The main idea is to focus on easing the GUI on multiple platforms to make accesible to C++ developers to develop and test path planning algorithms and be able to visualize its results in multiple ways.

## What would I like to add?

I would like to have multiple tools to test multiple problems, such as the solution on a dense graph from point A to point B. But this is seldom accepted by the nature or reality, objects and most importantly robots, have inherent dynamic properties that will have to deal with the algorithm solution. 

I would like to have a way of testing said algorithms with multiple layers of complexities, being able to add robot dynamics, diferrent space sampling and even some control logic to it. But for the time being, let me focus on creating a decent interface.

## What will I use?

I will use Qt6 for the GUI and the rest probably just C++. I would also like to make this buildable on multiple platforms, but that is a problem for the working project.
